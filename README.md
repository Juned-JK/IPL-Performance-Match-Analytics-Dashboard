# 🏏 IPL Performance & Match Analytics Dashboard

## 📌 Project Overview
This end-to-end Data Analytics project analyzes IPL cricket data from 2008 to 2026 to uncover insights related to team performance, player statistics, venue analysis, batting efficiency, bowling economy, and season-wise trends.

The project follows a complete analytics workflow:
**Data Cleaning (Python) → EDA (MySQL) → Visualization (Matplotlib & Seaborn) → Interactive Dashboards (Tableau & Power BI)**

---

**Data Processing**

## 📂 Project Workflow

**1️⃣ Data Collection**
- IPL Matches dataset (2008–2026)
- IPL Deliveries dataset (ball-by-ball data)

**2️⃣ Data Cleaning (Python + Pandas)**
- Merged matches and deliveries datasets
- Handled missing values and duplicates
- Feature engineering — Strike Rate, Economy Rate, Win %, Power Hitter metrics

**3️⃣ Exploratory Data Analysis (MySQL)**
- Season-wise match and run trends
- Team win percentage and total wins
- Top batsmen by runs, average, and strike rate
- Top bowlers by wickets and economy rate
- Venue-wise match distribution
- Power hitter analysis — Sixes, Fours, Strike Rate

**4️⃣ Visualization (Matplotlib & Seaborn)**
- Run distribution charts
- Season-wise trend analysis
- Player performance comparisons

**5️⃣ Dashboard Development**
Built interactive dashboards in:
- Tableau
- Power BI

## 📊 Key Insights
- **Gujarat Titans** lead with the highest win percentage (**60.9%**) while **Mumbai Indians** hold the most total wins (**152**)
- **V Kohli** is the all-time top run scorer with **8,598 runs** and **YS Chahal** leads wickets with **220 wickets**
- **Wankhede Stadium, Mumbai** is the most used IPL venue with **127 matches** hosted
- **AD Russell** dominates power hitting with a strike rate of **174** across **217 sixes** and **181 fours**
- **2023 season** recorded the highest wicket tally in IPL history
